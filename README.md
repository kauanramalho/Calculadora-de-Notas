Projeto em Python criado para facilitar o cálculo de notas no método de avaliação usado pela faculdade.
A ideia surgiu porque, no final do período, muitos alunos ficam confusos tentando calcular se foram aprovados direto, se ficaram de prova final ou quanto precisam tirar no próximo bimestre.

## Funcionalidades
O sistema funciona por menu no terminal e permite:

- Calcular a média entre o 1º e o 2º bimestre.
- Verificar se o aluno foi aprovado direto.
- Verificar se o aluno ficou de recuperação/prova final.
- Calcular quanto precisa tirar na prova final.
- Calcular quanto precisa tirar no 2º bimestre para ser aprovado direto.
- Listar o resumo das matérias já calculadas.

## Como funciona o cálculo
1 - Nome da Matéria
2 - Nota necessária para 2ºB
3 - Matérias
0 - Sair

Opção 1:
Calcula a média da matéria usando a nota do 1º e 2º bimestre.
O programa informa se o aluno foi aprovado ou se ficou de recuperação.

Opção 2:
Calcula quanto o aluno precisa tirar no 2º bimestre para ser aprovado direto.

Opção 3:
Mostra um resumo das matérias calculadas durante a execução do programa.

Opção 0:
Encerra o programa.

Validação de notas
O programa aceita apenas notas entre 0 e 10.
Caso o usuário digite uma nota inválida, o sistema pede para inserir novamente.
